## Current Features

These features are live and ready for you to use:

- **UI Cloning**: Generate pixel-perfect clones of website UIs from URLs or images.
- **Web Browsing**: Autonomously browse the web to use documentation or images as inspiration or context for your projects.
- **Public GitHub Integration**: Connect to public GitHub repositories to use existing codebases and save your code.
- **Chat History & Version Control**: Track your conversations and revert to previous project states.
- **Netlify Deployment**: Deploy your projects directly to Netlify, including support for custom URLs.

## Upcoming Features

We're actively working on expanding Same's capabilities. Here's what's next:

- **Full-Stack Support**: Build applications with backend logic, database connections, and more.
- **Private GitHub Integration**: Connect securely to your private GitHub repositories.
- **More Deployment Options**: Adding support for other hosting providers beyond Netlify.

We're excited about the future of Same and welcome your feedback and suggestions! Join our [community discord](https://discord.gg/tjHuQJaPsy) to share your thoughts.
